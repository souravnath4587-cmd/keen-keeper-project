# Keen Keeper Project

## Description

Keen Keeper is a modern and user-friendly web application designed to help users manage and organize important information efficiently. It focuses on simplicity, performance, and a clean user interface, making it easy for users to keep track of their data without unnecessary complexity.

---

## Technologies Used

- React.js
- Vite
- Tailwind CSS
- DaisyUI
- React Router

---

## Features

- Dynamic page routing with React Router
- Fast performance using Vite
- Clean and responsive UI with Tailwind CSS & DaisyUI
- Reusable component-based architecture
- Custom error handling page
- Detailed view pages (e.g., friend details or item details)

---
